My Projects

Java

I made a snake game using java language when we were taught java in our versity.This game is very simple having only one menu page.
The target is to acquire as much as points by eating bugs on the field through the snake.Keyboard button is used to controll the snake.
There is also a point system in the game which saves the highest score of the player and also shows the current score while playing.
The game ends if the snake hits any wall or bites itself.As the score increases the speed of the snake also rises.
